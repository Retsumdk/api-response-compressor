# api-response-compressor

Automatic gzip/brotli compression for API responses

## Features

- Production-ready code
- TypeScript/Python with full type safety
- Comprehensive error handling
- Built by Retsumdk

## Installation

```bash
git clone https://github.com/Retsumdk/api-response-compressor.git
cd api-response-compressor
bun install
```

## Usage

```bash
bun run src/index.ts --help
```

## Configuration

Create `config.json` (TypeScript) or `config.yaml` (Python) for custom settings.
